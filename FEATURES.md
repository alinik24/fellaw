# FelLaw - Features & Roadmap

## Current Features

### User Features
- **Urgent Help Workflow**: Quick access to emergency legal assistance
- **Case-Specific Intake Forms**: 
  - Family Law Inquiries
  - Employment Disputes
  - Consumer Protection
  - Traffic Violations
  - Immigration/Visa Issues
  - Housing Issues
- **Lawyer Finder**: Search and filter qualified legal professionals
- **Document Upload**: Upload and analyze legal documents with AI
- **Multi-language Support**: English and Spanish interface
- **Dark/Light Mode**: User preference theme switching

### Lawyer Features
- **Professional Dashboard**: Manage cases and client interactions
- **Onboarding System**: Complete registration and verification
- **Case Management**: Review and accept client cases
- **Document Review**: AI-assisted document analysis

### Technical Features
- **Voice Recording**: Accessibility feature for case intake
- **Responsive Design**: Mobile-first, works on all devices
- **AI-Powered Assistance**: OpenRouter Vision for document analysis
- **Secure Authentication**: JWT-based auth system
- **File Management**: Secure document storage and retrieval

## Technology Stack

**Frontend:**
- React 18 + TypeScript
- Vite (build tool)
- TailwindCSS + Radix UI
- React Router v6
- React Hook Form

**Backend:**
- FastAPI (Python)
- PostgreSQL + SQLAlchemy
- Alembic (migrations)
- JWT Authentication
- OpenRouter API

**AI Services:**
- OpenRouter Vision (Claude 3.5 Sonnet)
- Azure OpenAI (optional)
- Azure Document Intelligence (optional)
